We prefer Java as programming language, but you may also use C++ or 
if you ask us before hand then also another language of your choice.

Comments and identifier names should be in English.

The documentation can be browsed from .../doc/index.html and here you will
find the description of the binary file format in the class DataFile. 

= Students applying for a internship =

Please create a new package "com.ifta.candidate.test.YOURNAME".  

See the interfaces in the package "com.ifta.candidate.test.interfaces" and 
the documentation of the file format in the "DataFile" interface.

Implement a class that can read the binary file "data/test.bin" and creates 
a ASCII file as output with TAB (\t) as delimiter.  

For students it is not required to implement the interfaces and iterators.  
Also it is allowed to read the whole file into RAM.

Run the ANT task "candidate-test-result-zip.xml" to create a ZIP file with
the results and send it back to IfTA.

Please also tell us how long it took you to solve this test - depending on 
your skills we expect a "guru" to solve this in 1 hour - do not spend more
than 2 hours, just send us what you have after 2 hours. 

= Software Engineers applying for a regular job = 

Please create a new package "com.ifta.candidate.test.YOURNAME" and implement
the interfaces in the package "com.ifta.candidate.test.interfaces" to create 
a "DataFile" class that can read the binary file "data/test.bin". 

Use the interface methods to access the data and create plots of the 
resulting data in the program of your choice, e.g. create a ASCII file
and plot in Excel, GNU plot, etc. 

Analyze the plots - there are several oddities - point them out.  Please 
save the image of the plot and your notes in your package directory. 

Run the ANT task "candidate-test-result-zip.xml" to create a ZIP file with
the results and send it back to IfTA.

Please also tell us how long it took you to solve this test - depending on 
your skills we expect a "guru" to solve this in 1 hour and an alumnus in up 
to 5 hours.
